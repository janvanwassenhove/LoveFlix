# LoveFlix Desktop App - Assets

Place your app icons here:

- `icon.png` - PNG icon (512x512 recommended)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon

## Creating Icons

You can use tools like:
- [electron-icon-maker](https://www.npmjs.com/package/electron-icon-maker)
- [IconConverter](https://iconverticons.com/)

### Quick Command to Generate Icons
```bash
npx electron-icon-maker --input=icon.png --output=./
```

## Suggested Icon Design

Create a heart-shaped icon with film reel elements in Netflix red/pink gradient colors:
- Primary: #E50914 (Netflix Red)
- Secondary: #FF6B9D (Romantic Pink)
- Background: #141414 (Netflix Black)
